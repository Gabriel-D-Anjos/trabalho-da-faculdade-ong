#  ONG Exemplo

Este repositório contém o desenvolvimento do site institucional da **ONG Exemplo**, criado com foco em simplicidade, acessibilidade e responsividade.  
O objetivo é apresentar a missão da ONG, seus valores, projetos sociais, além de disponibilizar um formulário de cadastro para voluntários.

---

## ✅ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica e acessível  
- **CSS3** — Layout responsivo, grid e estilização moderna  
- **JavaScript** — Máscaras de input, validação de formulário e interação  
- **Git / GitHub** — Controle de versão e hospedagem  
- **GitHub Pages** — Deploy do site

---

## ✅ Funcionalidades do Projeto

### ✅ Navegação Responsiva
- Menu adaptável para telas menores  
- Abertura e fechamento via JavaScript  
- Acessibilidade com `aria-expanded`

### ✅ Seções do Site
- **Missão**  
- **Visão e Valores**  
- **Destaques de Projetos** (com imagens)  
- **Formulário de Cadastro/Voluntariado**  
- **Contato**

### ✅ Máscaras de Formulário
- Máscara para **CPF**  
- Máscara para **telefone**  
- Máscara para **CEP**  

### ✅ Validação
- Validação nativa com melhorias de acessibilidade  
- Foco automático no primeiro campo inválido  
- Mensagens de feedback ao usuário
